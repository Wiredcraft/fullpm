# KenHQ Client

## Setup & Running
##### Prerequisites
 - Node.JS & NPM

##### Install Steps
 - `npm install`

##### Running
 - `npm start`
 - `npm run serve`


 ##### production
  - `npm run build-prod` (Generate static assets)
  - `npm run run-prod` (Generate static assets, and serve)

##### Extra
If you need auto-refresh
 - Install `npm install browser-sync -g`
 - Run `npm run watch-dev`
